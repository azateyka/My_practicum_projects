
Исследовательский анализ данных экономического развития стран (1960 – 2024 годы) и его связи с другими показателями.
Выбраны страны из разных регионов у которых за последние десятилетия наблюдался активный рост экономического развития (Индия, США, Индонезия, Польша, Ю.Корея, Турция).
Требовалось собрать, обработать, проанализировать данные Всемирного банка (получение ключевых показателей через API) и проверить гипотезу о связи экономического роста и показателей развития общества.
